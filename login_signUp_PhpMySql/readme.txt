this is a free and open source Login-SignUp system built in phpMySql.

for install it follow the instruction below -->

1) download zip

2) open dbcon.php in text editor

3) change first four variable $dbhost, $username, $passowrd, $dbname according to your credentials

4) create database given in above step (skip if already done)

5) import logsign.sql file into database in phpMyadmin (Be carefull With your Existing Database)

6) now add this in your web project.

7) done
